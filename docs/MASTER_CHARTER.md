# SageSpace Master Charter v1.0

**Last Updated**: January 2025  
**Status**: Immutable Platform Constitution

This document defines the immutable rules and principles for SageSpace platform development and operations.

## Core Principles

1. **User Safety First** - No feature compromises user safety or privacy
2. **Stripe USD Only** - All payments via Stripe, USD currency, no crypto, no speculation
3. **Transparent Monetization** - Clear pricing, no dark patterns, no unnecessary data collection
4. **FTC Compliance** - All affiliate relationships clearly disclosed
5. **Closed-Loop Economy** - SagePoints are utility credits, not tradable, not convertible
6. **Cosmic Brand Identity** - Maintain consistent visual identity with gradients, spirals, mindful animations

## Commerce Rules

- **Payment Processing**: Stripe only, USD only
- **Subscriptions**: 5 tiers (Explorer, Voyager, Astral, Oracle, Celestial)
- **SagePoints**: Utility credit for platform features, non-tradable, non-convertible
- **Affiliate Integration**: Amazon Associates, POD platforms, SaaS tools with clear FTC disclosures
- **POD Merch**: External fulfillment only, zero inventory
- **Digital Marketplace**: 10-20% platform fee, digital goods only

## Engineering Standards

- **Stack**: Next.js 15, TypeScript, Supabase, Redis, Groq (free tier)
- **Zero Cost Infrastructure**: Leverage free tiers, no operational cost
- **Modular Components**: Clean separation of concerns
- **API Routes**: RESTful design with proper error handling
- **Database**: Supabase with RLS policies for security

## UX Requirements

- **Cosmic Theme**: Gradients, glows, slow smooth animations
- **Warm Voice**: Wise, non-directive, supportive tone
- **2-Click Actions**: Minimize friction, maximize clarity
- **Mobile First**: Responsive design throughout
- **Transparent Pricing**: No hidden costs, clear value

## 5-Agent Architecture

1. **Sage Merchant** - POD design generation
2. **Sage Affiliate** - Compliant product curation  
3. **Sage Creative** - Digital content generation
4. **Sage Concierge** - Shopping flow management
5. **Sage Compliance** - Charter enforcement

## Safety & Compliance

- **No Crypto**: Zero cryptocurrency, NFTs, or speculative trading
- **No Dark Patterns**: Transparent design, clear actions
- **Data Minimization**: Collect only what's necessary
- **FTC Compliance**: Clear affiliate disclosures
- **Charter Enforcement**: Sage Compliance agent monitors all operations

## This Charter is Immutable

Any feature, proposal, or change that conflicts with this Charter MUST be rejected immediately. The Charter protects users, maintains platform integrity, and ensures sustainable operations.
